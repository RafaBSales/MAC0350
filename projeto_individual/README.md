# Introdução
Este é um projeto individual de [MAC0350 (Introdução ao Desenvolvimento de Sistemas de Software)](https://uspdigital.usp.br/jupiterweb/obterDisciplina?sgldis=MAC0350)
que visa explorar os conceitos vistos nas primeiras semanas de aula do curso. O projeto consiste na **criação de um web app simples** que integra o back-end (com FastAPI, HTMX e SQL) e o front-end.

---

## Caderno de Receitas
O web app que será desenvolvido será um **caderno de receitas**, isto é, um arquivo de receitas que oferece ao usuário certas funções e serviços:

- Adicionar novas receitas (com nota para cada receita);
- Remover receitas;
- Atualizar detalhes das receitas;
- Atualizar as notas das receitas;
- Buscar por receitas.

Por enquanto, o web app terá **pelo menos** duas telas: uma para a busca de receitas e outra para a adição de uma nova receita.
